# google-serp-automation
Google Search Counter:
A Python automation project that analysis Google Search Engine Results Pages (SERPS) using a list of search queries stored in an Excel workbook. 

Overview:
This tool reads search queries from Excel, performs searches and extracts useful information from the results, and wrties the collected data back into the workbook. 

The project was created to practice Python automation, data processing, and working with structured datasets.

Features:
- Reads search queries from Exel
- Process multiple worksheets
- Perform automated searches
- Extract search results information
- Count sponsored search results
- Record timestamps for each search
- Write results back into Excel
- Preserve workbook formatting where possible

Technologies
- Python
- pandas
- openpyxl
- asyncio
- urlib
- Excel automation

Skills Demonstrated
- File I/O
- Data processing
- Python scripting
- Spreadsheet automation
- Asynchronous programming
- Error handling
- Data organization 

Motivation: 
This project is intended as a learning exercise focused on Python automation and data processing. It demonstrates practical scripting techniques rather than serving as a production-ready search engine analysis tool.
